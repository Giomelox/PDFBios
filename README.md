# Passo a passo para configuração do cloud run

## 1º Criar um projeto no ambiente do GCP

## 2º Acessar -> Cloud Run e Criar um serviço linkado com o GitHub, após selecionar o GitHub, configure o cloud build com as informações do repositório, e após isso verifique as configurações do build.

## 3º Após a criação do serviço com as devidas configurações, o GCP irá criar uma imagem docker e implantar no cloud run então após o deploy será disponibilizada uma URL, basta acessa-lá para confirmar que o ambiente está online.
